# ⚡ Calculadoras Elétricas em JavaScript

Projeto desenvolvido para fins de **aprendizado em um curso**, contendo algumas ferramentas simples relacionadas à área elétrica feitas em **JavaScript**.

O sistema utiliza **entrada de dados via `prompt`** e exibe os resultados no **console do navegador**.

---

## 📌 Funcionalidades

O projeto possui atualmente as seguintes funcionalidades:

### 🔌 1. Calculadora de Consumo Elétrico

Calcula o consumo mensal de um eletrodoméstico com base em:

* Nome do eletrodoméstico
* Potência (W ou kW)
* Tempo de uso diário
* Valor do kWh

Saída:

* Consumo mensal em **kWh**
* **Custo mensal estimado**

---

### 🔁 2. Conversor de Unidades Elétricas

Permite converter valores entre diferentes **prefixos do Sistema Internacional** usados em eletrônica:

* G (Giga)
* M (Mega)
* k (Kilo)
* Base
* m (Mili)
* µ (Micro)
* n (Nano)
* p (Pico)

| Arquivo                    | Função                                         |

| `index.html`               | Arquivo principal que carrega os scripts       |

## ▶️ Como Executar

1. Baixe ou clone o repositório:

```bash
git clone https://github.com/vondenryan/calculadora-eletrica.git
```

2. Abra o arquivo:

```
index.html
```

3. Abra o **Console do navegador** (`F12`).

4. O sistema exibirá um **menu de opções via prompt**.
